Qual a diferença entre classes e IDs no HTML?

Classes são reutilizáveis e permitem aplicar o mesmo estilo a múltiplos elementos. Use o seletor .classe no CSS.

IDs são únicos para cada elemento e devem ser usados uma única vez por página. Use o seletor #id no CSS. IDs também têm maior especificidade.
